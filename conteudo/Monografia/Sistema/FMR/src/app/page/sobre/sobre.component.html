Find My Republic

Trabalho de Monografia II do curso de Ciência da Computação da Universidade Federal de Ouro Preto, a ser defendido em Dezembro/2019.
O objetivo do sistema é recomendar aos estudantes universitários as repúblicas que mais possuem compatibilidade com a sua personalidade.

O Sistema

O objetivo do sistema é recomendar ao estudante as repúblicas que mais possuem compatibilidade com sua personalidade. A ideia é construir um software responsável por armazenar a personalidade dos calouros e das repúblicas, realizando a recomendação baseada na distância das personalidades.

O sistema é dividido em dois componentes, a depender do tipo do usuário que está acessando (Estudante ou República)

O produto é um sistema web que oferece serviços de relatórios de personalidade de estudantes e repúblicas obtidos a partir do Big Five Inventory. Também pode oferecer serviços de exibição de informações dos usuários, uma vez que eles permitam a exibição.

O sistema permite ao estudante consultar e visualizar dados que representam a personalidade das repúblicas. Essa representação diz respeito a agregação das personalidades dos que ali residem.

Privacidade

Para a utilização do sistema, é necessário realizar um cadastro. Os dados obrigatórios para realizar o cadastro são Email e Senha. Entretanto, é interessante que se preencha todos os dados para que se obtenha um melhor resultado do software.

Nenhuma informação será divulgada sem a autorização do respectivo usuário.

Os dados sigilosos são criptografados para serem armazenados no banco de dados.
